🏹 BG 3.27 | April 16, 2026

 

प्रकृतेः क्रियमाणानि गुणैः कर्माणि सर्वशः ।

अहङ्कारविमूढात्मा कर्ताहमिति मन्यते ॥ २७ ॥

 

prakṛteḥ kriyamāṇāni
guṇaiḥ karmāṇi sarvaśaḥ
ahaṅkāra-vimūḍhātmā
kartāham iti manyate

 

The spirit soul bewildered by the influence of false ego thinks himself the doer of activities that are in actuality carried out by the three modes of material nature.

 

Kṛṣṇa's instruction is dismantling surgery, not gentle contemplation. The verse operates like debugger revealing where bug lives: between kartā (doer) and aham (I). Bewildered soul inserts himself into transactions carried out by prakṛti's guṇas. This is diagnosis, not blame.

 

Sāṅkhya is precise. Prakṛti is material matrix — three-guṇa substance churning all manifestation. Puruṣa is pure consciousness, the witness — never actor, never acted upon. Between them stands ahaṅkāra — ego-function appropriating experience, transforming "awareness of action" into "I am acting." Only wrong in system: false watermark of identity stamped on actions and results.

 

Consider mechanism. You decide to raise arm. Nerve signals fire. Muscles contract. Arm rises. Ego says: "I raised my arm." But trace chain: where did decision emerge? From conditioning, from prāṇa, from guṇas in motion. Consciousness never moved — only witnessed what prakṛti enacted. This is vimoha — delusion conflating witnessing with authorship.

 

Implications devastate spiritual materialism. Cannot keep score of sādhana. Cannot claim "I meditated for hour" as achievement. Cannot accumulate spiritual credits. There is no accumulator. Only action happening, witnessed. Āsana becomes yoga only when practitioner stops performing and starts witnessing. Body moves. Breath moves. Mind observes. None of it is "mine."

 

On Amāvāsyā when moon dissolves and energies pull toward internalization, practice is radical: track every "I" that arises. I am doing this. I achieved that. I need this. I want that. Each one is ahaṅkāra-vimoha. Each is false attribution. Each is soul asleep, dreaming it is dreamer.

 

YOGA SŪTRA CONNECTION — DISSOLUTION THROUGH DISCIPLINE:

 

YS II.1: tapaḥ-svādhyāyeśvara-praṇidhānāni kriyā-yogaḥ — "Austerity, self-study, and surrender to Īśvara constitute the practice of yoga."

 

Patañjali offers antidote to ego-claim. Where BG 3.27 diagnoses disease — ahaṅkāra-vimoha — YS 2.1 prescribes cure: tapas burns identification, svādhyāya reveals puruṣa/prakṛti distinction, praṇidhāna redirects claim toward divine surrender.

 

Action performed as kriyā under sign of surrender rather than acquisition. Praṇidhāna cuts through vimoha by redirecting "I act" to "You act through me." Work with full competence and zero claim. Cashier counts millions but claims nothing. Yogī acts knowing prakṛti acts, not the self.

 

Bhakti-Yoga synthesis: "I am not the doer — You are." Root-level correction transforming every action into worship: prakṛti dancing for puruṣa's witness.

TERM DEFINITIONS:

prakṛti — "nature, matrix" — Primordial material substance; unconscious substrate containing three guṇas. Code analogy: Operating system, the platform upon which all apps run. Not conscious — appears conscious only when reflecting puruṣa's light.

guṇa — "strand, quality" — Three constituent qualities of prakṛti: sattva (lucidity), rajas (activity), tamas (inertia). All material phenomena are combinations of these three. Code analogy: Three basic system states — processing, clarity, stasis.

ahaṅkāra — "I-maker" (aham + kāra) — Ego-function creating identification, appropriating experience as "mine." Self-referential pointer mistaking traced memory address for source of consciousness. Code analogy: Recursive function calling itself until stack overflow.

vimoha — "delusion, bewilderment" — State of confusion by false identification. Confusion caused by ahaṅkāra making soul think it is doer. Code analogy: Process thinking it's kernel when actually just user-space application — complete misidentification of privilege level.

kartā — "doer, agent" — Subject performing action. In reality, only prakṛti through guṇas is agent. Soul claims role erroneously. Code analogy: User thinking they compiled code when compiler did all work — they just pressed return.

manyate — "thinks, imagines" — From man, root of mind. Ego's primary activity: mental fabrication, constructing narratives of authorship. Code analogy: Frontend displaying dashboard user thinks represents causation, when actually just logging events generated elsewhere.
